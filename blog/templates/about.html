关于本网站

该网站后端由基于 Python 的 Django 框架开发。因为本人对前端技术了解较少，所以基本的html和css参考了某位罗马尼亚大神 Github 的部分代码。此外，还用到了 Font Awesome 和 Bottons 这两个制作精美的开源库，用于页面中的部分图标和按钮，在此也强烈推荐。本站点目前托管在新浪 PaaS 平台 SAE 上。

对包括但不限于以上开源代码的开发者表示衷心感谢。
